KEY FEATURES:
✓ Gin web framework for high-performance HTTP routing
✓ Clean architecture with separated concerns
✓ Thread-safe in-memory store with sync.RWMutex
✓ RESTful API endpoints
✓ Zelda-inspired retro pixel art theme
✓ Animated UI with CSS animations
✓ Random fun purchase messages
✓ Statistics tracking (potions sold, gold earned)

API ENDPOINTS:
- GET  /                       # Main shop page (HTML)
- GET  /api/stats              # Get shop statistics
- POST /api/buy                # Purchase a potion (50 gold)
- POST /api/reset              # Reset statistics
- GET  /api/health             # Health check

TECHNOLOGIES:
- Go 1.21+
- Gin Web Framework (github.com/gin-gonic/gin)
- Pure HTML/CSS/JavaScript (no frontend framework)
- In-memory storage (no database required)

QUICK START:
1. cd /Users/benson.quach/benson/SANDBOX/Go/potion-shop-gin
2. make install    (or: go mod download)
3. make run        (or: go run main.go)
4. Open http://localhost:8080 in your browser

THREAD SAFETY:
The store uses sync.RWMutex to ensure thread-safe concurrent access.
Multiple users can purchase potions simultaneously without data races.
